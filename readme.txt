== Description ==

Remove links for parent-items, add a is-parent class and/or add the parent-item to the submenu.

== Installation ==

1. Upload `link-wp-functions.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change settings through Settings -> wp_list_pages tweaks
